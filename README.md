*Setup*

Prerequisites

Install Node.js (v16+) and React Native CLI.
npm install -g react-native-cli
Set up an OpenTelemetry Collector locally.
npm install
Start the app:
iOS: npx react-native run-ios
Android: npx react-native run-android
